## Solazah 是什么

Solazah 是一个住在你桌面上的智能助手(AI agent)。

它不是一堆工具的拼盘,而是一个能听懂你需求、主动替你动手的助手:你用一句话说出想做的事,它来调动你电脑上的应用、查找信息、代你操作,把事情办成。用得越久,它越记得住你的习惯和上下文,越懂你。

按一下快捷键(默认 `Alt + Space`)随叫随到,问完即走,不打断你的节奏。

## 能帮你做什么

- **跟它聊,问什么都行** —— 写东西、改文案、做总结、查资料、出主意,内置免费 AI 额度,开箱即用。
- **代你在应用里办事** —— 把跨应用的琐碎操作交给它,你说目标,它一步步替你完成。
- **一搜即达** —— 快速找到并打开应用、跑常用操作,不用来回翻找。
- **按需扩展** —— 剪贴板增强、OCR 文字识别、计算器、笔记…… 用插件给它加本事。
- **记得住你** —— 它会记住你的偏好和正在做的事,下次接着聊,不用从头解释。

## 意见反馈

用得不顺手、发现 bug、有新想法,都欢迎告诉我们 —— 你的每一条反馈都会进入**这个仓库**,以 Issue 形式被看到、被跟进。

### 两个入口,同一个收口

不管从哪个入口提交,反馈最终都会汇成本仓库的一条 Issue:

1. **应用内「意见反馈」窗口** —— 在 Solazah 里打开反馈窗口,填写表单提交。
2. **直接跟 Copilot 说** —— 在对话里说出你的反馈(例如"我要反馈个 bug""有个建议"),助手会帮你把内容整理清楚,并代为提交。

### 提交时可以填什么

- **反馈类型**:问题 / 建议 / 其他 —— 会作为 Issue 的标签,方便分类处理。
- **反馈正文**:描述你遇到的问题或想法(如果是 bug,最好带上复现步骤)。
- **联系方式**(选填):QQ / 微信 / 邮箱 / 手机,方便我们回复你。
- **是否附带诊断信息**(可选):勾选后会一并附上你的运行环境,帮我们更快定位问题。

### 提交之后

提交成功会返回这条反馈对应的 **Issue 链接**。你可以点开它**实时追踪处理进度**,也能在下面补充信息,或看到我们的回复。

## 反馈管理与提反馈须知

### 标签体系

每条反馈会被打上类型标签,便于分类与跟进:

| 标签 | 含义 |
| --- | --- |
| `问题` | 功能不工作、报错、卡死、结果不对 |
| `建议` | 希望新增或改进某个功能 |
| `其他` | 不属于以上的反馈 |

我们会逐条查看汇入的反馈,按标签分类跟进,问题类会优先排查。

### 怎么提一条"有用"的反馈

为了让我们更快帮到你,一条好反馈最好包含:

- **复现步骤**:你是怎么一步步操作、然后出问题的?
- **预期 vs 实际**:你本来期望发生什么,实际发生了什么?
- **版本与系统**:你用的 Solazah 版本、操作系统(勾选"附带诊断信息"即可自动带上)。
- **截图**:如果方便,贴一张会更直观。

## 隐私与数据说明

我们只收集帮助定位与改进所必需的信息,而且尽量把选择权交给你。

**诊断信息**(可选,默认附带,可随时取消)包含:

- **应用版本** —— 判断问题出在哪个版本。
- **操作系统与架构** —— 还原你的运行环境。
- **登录账号**(若你已登录) —— 便于必要时联系你、定位与你账号相关的问题。
- **设备 ID** —— 一个随机生成的标识,用于把这条反馈和该设备的错误日志关联起来,加速定位。它**不包含**你的姓名等个人身份信息。

**你的掌控权:**

- 不想附带上述任何环境信息?提交时**取消勾选"附带诊断信息"**即可,只发送你写的正文。
- 联系方式完全选填,你不填,我们就不会拿到。
- 反馈正文与附带信息会进入本仓库的 Issue,**请不要在反馈里填写密码、验证码、支付信息等敏感内容**。
